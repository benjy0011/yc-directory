# YC_Directory 

This is my Next.js learning repo.

Credits: [Tutorial Video](https://www.youtube.com/watch?v=Zq5fmkH0T78&t=19206s)

YC_Directory is a **Next.js** web application that allows users to post and pitch their startup ideas.  
It integrates with **Auth.js (NextAuth)** for authentication, **Sanity CMS** for content management, and **Sentry** for monitoring & issue reporting.  
GitHub OAuth is used for login.

---

## ✨ Features

- 🔑 Authentication with GitHub (via Auth.js)
- 📝 Startup posts/pitches stored & managed in Sanity
- 🕵️ Error tracking & feedback widget powered by Sentry
- ⚡ Built with Next.js (deployed on Vercel)

---

## 🛠️ Tech Stack

- Next.js — React framework
- Auth.js (NextAuth) — Authentication
- Sanity.io — Headless CMS
- Sentry — Error tracking & feedback
- Vercel — Deployment
